## 關於這個 Repo

這個 Git repo 是用來回報應用程式「[D&D Digital Character Sheet](https://dnd-character.fucod.com/)」使用上遇到的問題與建議。

如果對應用程式有任何問題或建議，請開一個 issue 讓我們可以追蹤與處理。在發佈內容前，請先搜尋是否有相關的議題。發文內容請保持友善與理性討論。請體諒因為人力因素無法立刻或是回覆所有的議題。感謝大家的配合。
